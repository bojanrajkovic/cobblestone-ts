# cobblestone-ts

> **Placeholder release.** `0.0.0` exists only to claim the package name and configure npm trusted publishing. Real releases start at `0.1.0`.

TypeScript implementation of the [C2SP chunked-encryption scheme](https://c2sp.org/chunked-encryption) (Cobblestone-128 / Cobblestone-256), ported from the Go reference implementation [filippo.io/cobblestone](https://github.com/FiloSottile/cobblestone).

Universal (browser + Node + Deno/Bun/Workers), WebCrypto-only, zero runtime dependencies. Under construction.
